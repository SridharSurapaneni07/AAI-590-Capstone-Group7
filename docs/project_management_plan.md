## Final Title
BharatPropAdvisor: A Pan-India Multimodal Transformer-Based Recommender System for Personalized Property Investment Decisions with Maximum ROI Prediction

## Timeline & Milestones

The project will be executed over a 7-week timeline, adhering strictly to the capstone milestones:

### Week 1 (Module 1): Project Initiation
- **Goals:** Form the team, select topic and datasets.
- **Deliverables:** Draft Introduction, Draft Dataset Summary.
- **Status:** Completed.

### Week 2 (Module 2): Planning & Setup
- **Goals:** Finalize project management plan, initialize GitHub repository.
- **Deliverables:** Streamlit app wireframe, Project Management Plan, GitHub setup.
- **Status:** In Progress.

### Week 3 (Module 3): Data Preparation
- **Goals:** Data cleaning and Exploratory Data Analysis (EDA) on primary dataset.
- **Deliverables:** Dataset Summary final draft, Data Visualizations (e.g., price distribution, correlation heatmap).
- **Status:** Pending.

### Week 4 (Module 4): Modeling Strategy
- **Goals:** Finalize model selection, complete literature/background research.
- **Deliverables:** Literature/Background draft, Colab Notebooks setup, MLflow/Optuna configuration.
- **Status:** Pending.

### Week 5 (Module 5): Training & Baseline Methods
- **Goals:** Train baseline models (XGBoost, MLP), begin fine-tuning Vision Transformer (ViT-base) on secondary dataset.
- **Deliverables:** Methods draft, initial training curves.
- **Status:** Pending.

### Week 6 (Module 6): Full Multimodal Fusion & App
- **Goals:** Complete fine-tuning of IndicBERT/mBERT (Text) and FT-Transformer/TabNet (Tabular). Build and train custom cross-attention Fusion layer. Complete Streamlit deployment.
- **Deliverables:** Results section started, Streamlit App (Input form + dashboard).
- **Status:** Pending.

### Week 7 (Module 7): Final Deliverables
- **Goals:** Present final live app, record video presentation, complete written report.
- **Deliverables:** 20-page report, 10-12 min recorded presentation (minimum 8 visuals), Live App Demo, Peer Evaluation, GitHub cleanup.
- **Status:** Pending.
